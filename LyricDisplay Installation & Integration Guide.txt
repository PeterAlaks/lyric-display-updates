LYRICDISPLAY - INSTALLATION & INTEGRATION GUIDE
=================================================================

Professional real-time lyric display for streaming, church services and other live events
Version: 4.0.1 | Windows 10/11 (64-bit)

-----------------------------------------------------------------
WHAT IS LYRICDISPLAY?
-----------------------------------------------------------------

LyricDisplay is a free desktop app that provides transparent lyric overlays for OBS Studio and VMix. Perfect for:
- Church worship services
- Live concerts & events  
- Karaoke streaming
- Multi-language presentations

Key Features:
- Dual independent outputs with transparent backgrounds
- Real-time sync across displays
- Translation support (bracketed lines)
- 13 professional fonts with full styling controls
- Keyboard-driven workflow
- Secondary mobile/web controllers with join-code security
- Auto-updates via GitHub releases

-----------------------------------------------------------------
SYSTEM REQUIREMENTS
-----------------------------------------------------------------

Minimum:
- Windows 10/11 (64-bit)
- 8 GB RAM
- Dual-display capable GPU
- 500 MB disk space

Recommended:
- 16 GB RAM
- Dedicated GPU
- Dual monitors for control + preview

-----------------------------------------------------------------
INSTALLATION
-----------------------------------------------------------------

1. Download Latest Release
   Go to: https://github.com/PeterAlaks/lyric-display-updates/releases/latest
   Download: LyricDisplay-Setup.exe

2. Run Installer
   - Right-click installer → Run as Administrator
   - If SmartScreen appears: More info → Run anyway
   - Follow wizard, accept defaults
   - Create desktop shortcut (recommended)

3. Launch Application
   - Open from desktop shortcut or Start menu
   - Control panel will appear

-----------------------------------------------------------------
OBS STUDIO INTEGRATION (SAME COMPUTER)
-----------------------------------------------------------------

STEP 1: Add Browser Source

1. In your OBS scene, click [+] in Sources panel
2. Select "Browser"  
3. Name it "Lyrics - Output 1"
4. Click OK

STEP 2: Configure Browser Source

Enter these exact settings:

URL: http://localhost:4000/#/output1
Width: 1920
Height: 1080
FPS: 30
☑ Shutdown source when not visible
☑ Refresh browser when scene becomes active

Click OK to save.

STEP 3: Configure LyricDisplay

1. In LyricDisplay Settings Panel, select "Output 1"
2. Basic settings to start:
   - Font: Montserrat or Open Sans
   - Font Size: 48-60 px
   - Font Color: #FFFFFF (white)
   - Drop Shadow: Opacity 5-7
   - Position: Lower Third

3. Click "Preview Output 1" to test

STEP 4: Test Connection

1. Load a test lyric file in LyricDisplay
2. Click any lyric line
3. Check OBS - lyrics should appear
4. Toggle "Display Output" switch to hide/show

FOR OUTPUT 2 (OPTIONAL):
Repeat steps above but use:
- Source name: "Lyrics - Output 2"  
- URL: http://localhost:4000/#/output2

-----------------------------------------------------------------
NETWORK SETUP (LYRICDISPLAY ON SEPARATE PC)
-----------------------------------------------------------------

Use this when running LyricDisplay on one computer and OBS on another.

REQUIREMENTS:
- Both PCs on same network (LAN/WiFi)
- Ethernet recommended for stability

STEP 1: Set Static IP on LyricDisplay Computer

Windows Instructions:
1. Press Win+R, type: ncpa.cpl, press Enter
2. Right-click active network adapter → Properties
3. Double-click "Internet Protocol Version 4 (TCP/IPv4)"
4. Select "Use the following IP address"
5. Enter (adjust to your network):
   
   IP address: 192.168.1.100
   Subnet mask: 255.255.255.0
   Default gateway: 192.168.1.1
   Preferred DNS: 8.8.8.8

6. Click OK → OK → Close

STEP 2: Verify IP Address

On LyricDisplay PC:
1. Press Win+R, type: cmd, press Enter
2. Type: ipconfig
3. Confirm your static IP shows under your adapter

STEP 3: Configure OBS Browser Source

On your OBS computer, use this URL in Browser Source:

http://192.168.1.100:4000/#/output1

(Replace 192.168.1.100 with your LyricDisplay PC's static IP)

STEP 4: Test Connection

On OBS computer:
1. Open web browser
2. Navigate to: http://192.168.1.100:4000
3. Should connect successfully
4. If error, check firewall settings below

FIREWALL FIX (If connection fails):

On LyricDisplay computer:
1. Search for "Windows Defender Firewall"
2. Click "Allow an app through firewall"
3. Click "Change settings" → "Allow another app"
4. Browse to: C:\Program Files\LyricDisplay\LyricDisplay.exe
5. Check both "Private" and "Public"
6. Click OK

-----------------------------------------------------------------
SECONDARY CONTROLLERS (MOBILE / WEB)
-----------------------------------------------------------------

1. In LyricDisplay go to File > Connect Mobile Controller (or click the shield icon) to show the QR code and current 6-digit join code. The code resets when the app restarts.
2. On the secondary device (same network) scan the QR code or visit: http://<control-pc-ip>:4000/?client=mobile and enter the join code.
3. After pairing, the mobile layout can trigger lyric lines, toggle the display, run manual sync, and send lyric drafts for approval on the desktop.

-----------------------------------------------------------------
LOADING LYRICS
-----------------------------------------------------------------

METHOD 1: Load File
- File → Load Lyrics File (Ctrl+O)
- Browse to .txt or .lrc file
- Click Open

METHOD 2: Drag & Drop
- Drag .txt file from File Explorer
- Drop into LyricDisplay main panel

METHOD 3: Create New
- File → New Song (Ctrl+N)
- Type/paste lyrics
- Ctrl+T to add translation lines
- Click "Save and Load"

LYRIC FILE FORMAT:

First verse line
[Translation in brackets]

Second verse line
Another line

Chorus line
[Chorus translation]

Lines in brackets [], (), or {} are treated as translations.
Blank lines separate sections.

-----------------------------------------------------------------
TROUBLESHOOTING
-----------------------------------------------------------------

PROBLEM: Browser Source is Black/Empty

Solutions:
→ Confirm LyricDisplay is running
→ Verify URL is exactly: http://localhost:4000/#/output1
→ Right-click browser source → Refresh
→ Restart both LyricDisplay and OBS
→ Check Windows Firewall isn't blocking port 4000

PROBLEM: Network Connection Not Working

Solutions:
→ Verify both PCs on same network
→ Test ping: ping 192.168.1.100 (from OBS PC)
→ Temporarily disable firewall to test
→ Use http:// not https:// in URL
→ Check router isn't blocking local traffic

PROBLEM: Lyrics Not Updating in Real-Time

Solutions:
→ Click "Sync Outputs" in LyricDisplay settings
→ Refresh browser source in OBS
→ Check Socket.io connection (green indicator)
→ Restart LyricDisplay

PROBLEM: OBS Performance Issues

Solutions:
→ Enable "Shutdown source when not visible"
→ Lower browser source FPS to 30
→ Close unused preview windows in LyricDisplay
→ Check CPU usage
→ Move LyricDisplay to dedicated PC

PROBLEM: Styling Changes Not Applying

Solutions:
→ Use "Sync Outputs" button in settings
→ Refresh browser source in OBS
→ Check browser console (F12) for errors

-----------------------------------------------------------------
OBS TIPS & TRICKS
-----------------------------------------------------------------

1. POSITIONING
   LyricDisplay handles positioning internally via settings panel.
   No need to transform in OBS unless you have specific needs.

2. LAYERING
   Place browser source above your main video/camera layers.
   Transparency automatically works.

3. SCENE TRANSITIONS
   Use OBS scene transitions normally.
   Enable "Shutdown source when not visible" for better performance.

4. MULTIPLE OUTPUTS
   Use Output 1 for main broadcast language
   Use Output 2 for alternate language or in-house display
   Both can run simultaneously with different styling

5. BACKUP STRATEGY
   Keep lyrics folder organized by date/service
   Export setlists for recurring events
   Test connection before going live

6. PERFORMANCE
   Browser sources use CPU - monitor usage
   Consider dedicated GPU for encoding
   Close preview windows when not needed

-----------------------------------------------------------------
ADVANCED: COLOR CORRECTION IN OBS
-----------------------------------------------------------------

If you need to adjust lyric appearance in OBS:

1. Right-click browser source → Filters
2. Add "Color Correction" filter
3. Adjust:
   - Gamma for brightness
   - Contrast for boldness
   - Saturation for color intensity

Note: Better to adjust in LyricDisplay settings first.
